Setup in drei Blöcken

Ollama liefert eine einfache lokale API, Open WebUI stellt die Chat-Oberfläche bereit, und Cloudflare Tunnel bindet lokale Dienste ohne Port-Forwarding an eine Domain an. [web:5][web:19][web:9]

1 · Backend

Ollama installieren

Kleines Modell laden und direkt testen.

curl -fsSL https://ollama.com/install.sh | sh
ollama pull qwen2.5:3b
ollama run qwen2.5:3b "Hallo Chromebook!"
2 · Chat UI

Open WebUI starten

Python-Umgebung anlegen und Open WebUI lokal starten.

python3 -m venv venv
source venv/bin/activate
pip install --upgrade pip
pip install open-webui
open-webui serve
3 · Externer Zugriff

Cloudflare Tunnel

Hostnames für API und Chat im Dashboard definieren.

ollama.example.de → http://localhost:11434
chat.example.de   → http://localhost:8080
sudo cloudflared service install <TOKEN>
